# The Odin Project - Soluções em Ruby

Este repositório contém soluções e exercícios práticos do [The Odin Project](https://www.theodinproject.com/paths), um currículo abrangente para aprender desenvolvimento web.

## Sobre The Odin Project

The Odin Project é um currículo gratuito e de código aberto que ensina desenvolvimento web full-stack. Ele oferece um caminho de aprendizado estruturado com projetos e exercícios do mundo real.

## Estrutura do Repositório

Este diretório contém exercícios e projetos em Ruby concluídos como parte do currículo do The Odin Project.

- `ceaser_cypher.rb` - Implementação da Cifra de César

## Como Começar

Para executar qualquer script Ruby neste repositório, certifique-se de ter Ruby instalado em seu sistema:

```bash
ruby nome_do_script.rb
```

Visite [The Odin Project Paths](https://www.theodinproject.com/paths) para explorar o currículo completo e acompanhar seu progresso.
